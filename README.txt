SRL Reflection Library
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This is a heavily revised version of the current SRL Reflection Library.

This software is written to be used with Simba[1] and SMART[2].
This library also requires the SRL Repository[3]. More information
can be found at the Villavu forums[4].
